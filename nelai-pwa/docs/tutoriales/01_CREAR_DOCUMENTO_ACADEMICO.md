# Tutorial: crear un documento académico

## Objetivo

Crear un documento para **ensayo / propuesta / investigación** con el agente en modo **Académico**.

## Pasos

1. Ve a **Documentos** → **Crear documento**.
2. Si aparece **“Wallet bloqueada”**, pulsa **Desbloquear** y autentícate.
3. Elige tu **cuenta autora** (Substrate).
4. En “Perfil del agente”, selecciona:
   - **Académico (ensayos e investigación)**
5. Crea el documento y empieza a escribir.

## Consejo de estructura (rápido)

Para un ensayo típico:

- Título
- Introducción (tesis + contexto)
- Argumentos (2–4 secciones)
- Contraargumento (opcional)
- Conclusión
- Referencias (si aplica)
